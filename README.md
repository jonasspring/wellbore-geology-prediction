# Wellbore Geology Prediction (Rogii Kaggle Challenge)

An advanced machine learning project to predict geological properties/formations in horizontal subsurface segments using wellbore logging sensor data. 

## Project Overview
This repository contains my solution for the Kaggle challenge hosted by Rogii. The goal is to leverage data-driven models to predict subsurface geology from horizontal well segments. Accurate subsurface prediction is a key component in modern geosteering, helping operator teams optimize drilling paths and reduce risks in real-time.

**Reference:** 
Igor Kuvaev, Rafael Aguilar, John Granmayeh, Ryan Holbrook, María Cruz, and Ashley Oldacre. ROGII - Wellbore Geology Prediction. https://kaggle.com/competitions/rogii-wellbore-geology-prediction, 2026. Kaggle.

*Keywords: Geosteering, Well Logging, Subsurface Modeling, Time-Series/Depth-Series Analysis, Machine Learning Regression.*
